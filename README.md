# SVV Using Selenium 🧪🚀

## 📌 Project Overview

SVV Using Selenium is a repository created to maintain and organize Selenium automation programs, Software Validation and Verification (SVV) lab exercises, semester practice codes, and automation testing implementations developed during my academic journey.

This repository focuses on practical implementation of software testing concepts using Selenium WebDriver. It includes browser automation scripts, testing exercises, and validation workflows that help in understanding real-world software testing and automation processes.

## 💡 Idea Behind the Repository

The purpose of creating this repository is to keep all Software Validation and Verification practice programs and Selenium automation codes in one structured and accessible place.

Throughout the semester, multiple automation tasks and testing exercises were performed as part of academic learning and self-practice.
